# Modding Guide

Membuat kasus baru di RetroSleuth sangat mudah:

1. Buat folder baru di `/cases/`, contoh `my_new_case`.
2. Buat `case.json` untuk manifest kasus Anda.
3. Buat file `.md` untuk briefing dan bukti-bukti.
4. Tambahkan entry kasus Anda di `cases/index.json`.

Struktur JSON dan skema dapat dilihat di PRD bagian 10.
